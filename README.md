# parking-detector-final-project-DIP
AI-powered Smart Parking Detector using YOLOv11 for real-time campus occupancy monitoring. It detects vacant and occupied spots via bird’s-eye view. Features a Flask dashboard that integrates a high-speed live stream for a low-cost, scalable Florida Poly parking solution.

Two models were independently trained using different datasets annotated with labels for occupied and empty spots, and divided into different training splits. We tested both models on parking lot images taken at Florida Polytechnic University to demonstrate which model performs best. Model performance was evaluated using varying metrics, such as precision, recall, F1-score, and mAP50. Additionally, we used the confusion matrices as supporting test result visuals.
